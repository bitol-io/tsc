## Motivation

In many data platforms there exists an map data type.

> Example Databricks

```
CREATE TABLE product_properties_map_demo (
  product_id INT,
  product_properties MAP<STRING, STRING>,
  meta_data MAP<STRING, INT>
)
USING DELTA;
```

> Example Snowflake

```
CREATE OR REPLACE TABLE my_table_with_map_column (
  id VARCHAR,
  my_map MAP(VARCHAR, VARCHAR)
);
```
