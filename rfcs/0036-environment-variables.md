# RFC-0036: Environment Variables in Servers

**Superseded by [RFC-0050: Variables](0050-variables.md) and archived.**

This RFC has moved to:

- [`archive/0036-environment-variables.md`](archive/0036-environment-variables.md) — original proposal, kept for history.

RFC-0050 keeps a single idea — `${VAR_NAME}` interpolation — and extends it to any string value in the contract, dropping the `$var` annotation alternative, the `variables` declaration block, and parameterized `$import` that this RFC explored. New discussion should happen on RFC-0050.
