# RFC-0028: Deprecated flag for schema and properties

RFC-0028 is a shared RFC: the same `deprecated` boolean flag is added to both ODCS (v3.2.0) and ODPS (v1.1.0). It is maintained as a single document under ODCS v3.2.0.

See [`../odcs-v3.2.0/0028-deprecated-flag.md`](../odcs-v3.2.0/0028-deprecated-flag.md) for the canonical specification, including the applicable levels (schema objects and properties, including nested properties).

Approved by the TSC on 2026-06-30 for inclusion in ODPS v1.1.0.
