# RFC-0038: Context Block for AI and Semantic Interoperability

RFC-0038 is a shared RFC: the same `context` block is added to both ODCS (v3.2.0) and ODPS (v1.1.0). It is maintained as a single document under ODCS v3.2.0.

See [`../odcs-v3.2.0/0038-context.md`](../odcs-v3.2.0/0038-context.md) for the canonical specification, including the ODPS-specific applicable levels (Data Product, Output port — input ports refer to the linked ODCS contract).

Approved by the TSC on 2026-05-19 for inclusion in ODPS v1.1.0.
