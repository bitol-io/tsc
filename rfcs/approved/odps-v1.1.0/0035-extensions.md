# RFC-0035: Vendor Attribution for Custom Properties

RFC-0035 is a shared RFC: the same optional `vendor` field is added to the shared `customProperties` structure across ODCS (v3.2.0), ODPS (v1.1.0), and OORS (v1.0.0). It is maintained as a single document under ODCS v3.2.0.

See [`../odcs-v3.2.0/0035-extensions.md`](../odcs-v3.2.0/0035-extensions.md) for the canonical specification.

Approved by the TSC on 2026-06-30 for inclusion in ODPS v1.1.0.
