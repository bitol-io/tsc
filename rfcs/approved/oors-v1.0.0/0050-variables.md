# RFC-0050: Variables

RFC-0050 is a shared RFC: the same `${VAR_NAME}` interpolation mechanism (including the POSIX `${VAR_NAME:-default}` form) applies to every Bitol standard. It is maintained as a single document under ODCS v3.2.0.

See [`../odcs-v3.2.0/0050-variables.md`](../odcs-v3.2.0/0050-variables.md) for the canonical specification.

Approved by the TSC on 2026-07-13 for inclusion in OORS v1.0.0.
